Black Jane is a clean SuevaFree child theme with an optional slideshow on homepage.

== Description ==

Black Jane is a clean SuevaFree child theme with an optional slideshow on homepage, a new header layout in addition to the five header layouts available on SuevaFree and new Google Fonts

Created by ThemeinProgress, http://www.themeinprogress.com

Black Jane WordPress Theme is a child theme of SuevaFree Theme, Copyright 2025 ThemeinProgress.
Black Jane is distributed under the terms of the GNU GPLv3.
